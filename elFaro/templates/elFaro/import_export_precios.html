Importar / Exportar Lista de Precios

¿Cómo utilizar esta sección?

  1. Descarga tu respaldo actual desde Exportar lista.
  2. Para CSV, prepara un archivo con las columnas obligatorias: codigo_barras, nombre, precio. Opcionales: precio_vecino, sku.
  3. Para Bsale, sube la lista base y la lista vecino en formato .xls. La base aporta el precio normal y el vecino aporta el precio vecino.
  4. Escoge si deseas borrar todo y volver a cargar (recomendado) o solo sobrescribir productos existentes.
  5. Selecciona el archivo correspondiente y presiona el botón de importación. Confirma la acción cuando se solicite.
  6. Revisa los mensajes de resultado para conocer filas importadas y posibles errores.
